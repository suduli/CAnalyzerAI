This demo shows the visit counter functionality that has been added to CAnalyzerAI.

Click "Show Stats" to see current visit data...

Features:

  • Total Visits: Counts every page load
  • Unique Visits: Counts unique browser sessions
  • Persistent Storage: Data persists across browser sessions
  • Interactive Display: Click the counter in the footer to see details
  • Responsive Design: Works on mobile and desktop
  • Theme Support: Adapts to light/dark themes

How it works:

  1. The counter automatically tracks visits using localStorage
  2. Unique visits are determined by browser sessions (sessionStorage)
  3. The counter displays in the footer of the main site
  4. Click the counter to expand and see additional details
